# GlowTips Frontend

Next.js (App Router) TypeScript client for marketing pages, auth, dashboard, public tip pages, and OBS overlay.

## Features

- Tailwind + shadcn-style component primitives
- React Hook Form + Zod validation
- TanStack Query API integration
- Recharts wallet chart and fee breakdown
- Framer Motion overlay animations
- Dark/light mode support
- Global toasts + loading skeletons
- Realtime overlay listening via Socket.io

## Environment

- `NEXT_PUBLIC_API_URL=http://localhost:5000`
- `NEXT_PUBLIC_WS_URL=http://localhost:5000`
